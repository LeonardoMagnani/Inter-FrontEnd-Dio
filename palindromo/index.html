Os dados estão no console.

Esse foi um exercício para compreender o funcionamento de funções e manipulação de dados através do palíndromo e um substituidor de números pares.


Para iniciar a função abra o console (F12) e chame-as com a seguinte sintaxe:


1. palindromo('palavra')

2. subPares([1, 2, 3, 3, 2, 1])
